Заказ Лира (2026-07-14). Строить инкрементами, ПОСЛЕ КАЖДОГО шага сайт обязан работать.
Раздел делаем максимально ИЗОЛИРОВАННО: отдельная страница `queue.html` + отдельный
бэкенд-модуль `api_queue.py` + отдельные таблицы (префикс `queue_`). В существующие
страницы — минимум правок (только кнопка-вход над таблицей).

## Полная спека (источник истины)

### Вход в раздел
- Над таблицей Доблести, СПРАВА ВВЕРХУ — картинка торговца, указывает пальцем,
  приглашает войти. Клик → окно входа.
- Окно входа:
  1. Ввод игрового ника. По мере ввода — автоподсказки из РЕЕСТРА + текущего списка
     Доблести (чтобы выбрал свой ник).
  2. Ввод ОБЩЕГО пароля (лежит внизу окна «Список участников гильдии», кнопка G в игре;
     задаёт админ, хранить в queue_config).
  3. После верного ника (из списка) + общего пароля → предложить указать почту
     (резервное восстановление пароля) и придумать ЛИЧНЫЙ пароль. Далее входит только
     по личному паролю.
  4. Запоминать устройство/IP → «оставаться в системе» (device-токен в куке + БД).

### Твины / мэйн
- Ник в таблице, у кого в ТИТУЛЕ стоит `~Мэйн~` — значит это ТВИН, а мэйн указан в титуле.
  Пример: `ЧуДная` с титулом `~Шлюпка~` → мэйн = `~Шлюпка~`, ЧуДная — твин.
- У человека много твинов. Принцип общий для всех. Данные видны из таблицы/реестра.
- Аккаунт очереди привязывается к МЭЙНУ (все твины → один аккаунт).

### Очередь
- Раздел «Очередь за ресурсами с КХ». 3 очереди (обычные / редкие R / легендарные S),
  идут ПАРАЛЛЕЛЬНО, слева-направо. В начале каждой (слева) — кнопка «Встать в очередь».
- Можно встать одновременно в 3 очереди. В ОДНУ и ту же — не дважды: повторно встать
  можно только когда дойдёт очередь и заберёшь ресурс; потом снова встаёт с конца.
- Визуал: 3 пиксельные очереди зигзагом, человечки двигаются, над головой — ник.
  В конце каждой — торговец с будкой, раздаёт ресы своей категории.
  Референс: `Downloads/Gemini_Generated_Image_dxlgo9dxlgo9dxlg.png` (черновой).
  3 будки: Ларёк Расходников (обычные), Ларёк Редких (R), Ларёк Элиты (S/легендарные).

### Модельки (персонажи)
- Класс берётся из таблицы. Общая моделька по классу из `Downloads/Готовые 2D (общие)`,

### Ресурсы и распределение (файл `Награды за все этапы кх PW.txt`)
- 7 этапов, у каждого свой набор. Категории и стаки:
  ОБЫЧНЫЕ (нужно ≥60 доблести):
    Камень доблести (стак 50), Метеорит (стак 20), Знак единства (стак 3),
    Пилюля звёздного духа 4ур (стак 20); Жемчужина Фу Си / Колода карт /
    Камень бессмертных — всё пачкой первому в очереди.
  РЕДКИЕ R (нужно ≥100): Запечатанная грамота Лиги (по 2), Приказ Феникса (стак 50).
  ЛЕГЕНДАРНЫЕ S (нужно ≥100): Драконья чешуя (по 1), Сущность карты (по 1),
    Огненный цилинь (по 1), Высший камень божества (по 1).
- Топ-1/2/3 недели: первыми выбирают ОБЫЧНЫЙ ресурс — один любой стак указанного размера.
- Проводники (в Доблести `@darkero4ek` и `Мortаlitу`): ДО распределения по 10% камней
  доблести и 10% метеоритов КАЖДОМУ.
- Еженедельно Лир в программе сбора данных указывает, сколько этапов КХ закрыли →
  программа считает суммарные ресурсы и кому сколько выдать. Лишнее — распределяется в клане.
- Вставший в очередь может указать: на какого ТВИНА скинуть ресурс ИЛИ супругу передать.
